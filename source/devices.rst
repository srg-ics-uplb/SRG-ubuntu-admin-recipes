=================
Devices 
=================

* View CPU information.

.. code-block:: console

   $cat /proc/cpuinfo

* View memory information.

.. code-block:: console

   $cat /proc/meminfo

* View memory area used by I/O devices.

.. code-block:: console

   $sudo cat /proc/iomem

* Is my CPU 64-bit?Does it support virtualization?

.. code-block:: console

   $lscpu

* View PCI devices.

.. code-block:: console

   $lspci

* View devices.

.. code-block:: console

   $lshw
